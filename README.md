# Tournament Tracker

Webová aplikace pro správu a sledování turnajů ve stolním tenisu.

## Struktura projektu

- `frontend/` - React frontend aplikace

## Instalace a spuštění

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Aplikace běží na `http://localhost:5173`

## Technologie

- React 19
- Vite
- React Router
